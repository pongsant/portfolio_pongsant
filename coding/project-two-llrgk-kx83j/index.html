Web Design & Coding

While working on this project, I learned more about interactive design, how to organize web pages, and how to make websites look good on different devices. I also figured out how to add animations and interactive features using p5.js, which made the website more fun to use. This project helped me bring together my love for design and technology and made me better at web development.

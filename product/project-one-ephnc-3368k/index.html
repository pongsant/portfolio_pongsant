Interactive Table

Interactive Table

Our group project tackled loneliness by designing an interactive table with chess and tic-tac-toe engraved on its flip-top surface. I focused on the table’s design, using plywood for durability and engraving game patterns for a stylish, functional look. The legs symbolized modern connections, while matching stools with phone pockets encouraged face-to-face interactions.

We balanced aesthetics with functionality to make the table welcoming and accessible for public spaces. This project showed how design and teamwork can bring people together, proving that good design enhances real-world connections.

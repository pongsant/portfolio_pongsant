This project is a poster series celebrating four women type designers: Veronika Burian, Julieta Ulanovsky, Zuzana Licko, and Pooja Saxena. Each poster focuses on the designer’s name and biography, using typography as the main visual element.

The design explores repetition, scale, and layering of letterforms to show how type can create both meaning and visual texture. The background is made from overlapping letters, while the large names create hierarchy and identity for each poster. The goal of the project was to connect research about each designer with a strong visual expression.

Skills: Typography, Layout, Poster Design
Tools: Adobe Illustrator, Adobe InDesign
